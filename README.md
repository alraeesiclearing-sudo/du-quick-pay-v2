# du Quick Pay - Clone

This is a pixel-perfect clone of the du Quick Pay website.

## Features

- Responsive design
- Arabic language support
- Payment integration
- Multi-platform support

## Deployment

Deployed on Render at: https://du-quick-pay-v2.onrender.com

## Files Structure

- `/ar/` - Arabic language pages
- `/static/` - CSS, JavaScript, and assets
- `/src/` - Source files
- `manifest.json` - PWA manifest
